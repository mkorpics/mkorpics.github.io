RanchLife

A travel review web application

Skills: Code-first MVC, HTML5, CSS3, Bootstrap

Constructed an application using multiple models and foreign key relationships. Designed a general homepage with site descriptions and styled my reviews index page with corresponding detail, delete and edit pages.

This application naturally progressed from the previous static-website assignment, using the same ranch theme.
